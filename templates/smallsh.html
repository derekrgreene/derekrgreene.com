Details

This program implements a simple shell that can run commands in the foreground and background, with input and output redirection. It also handles built-in commands such as exit, cd, and status. Signals such as SIGINT, SIGCHLD, and SIGTSTP are handled as well. Comments can be added to the input by starting the line with '#'. Blank lines are ignored. The '&' character can be used to run a command in the background. Program will terminate upon 'exit' command.
