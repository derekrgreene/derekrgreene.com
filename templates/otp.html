Details

This project consists of 5 programs that operate to form a One-Time-Pad; keygen creates a pseudo-encrypted key of a specified length, enc_server is an encryption server that accepts TCP socket connections and receives plaintext and key data from a client enc_client. Upon encrypting the plaintext, it is sent back to the enc_client and output to stdout. enc_server will only accept connections from enc_client. dec_server is a decryption server that reads ciphertext and key data from client dec_client, decrpytes the ciphertext, and sends it back to dec_client. dec_client outputs the decrypted text to stdout. dec_server will only accept connections from dec_client. Supported characters: [A-Z] and space.
